Stack Data Structure

A Stack is a linear data structure that follows the LIFO (Last-In-First-Out) principle. It contains only one pointer, known as the top pointer, which points to the topmost element of the stack. Whenever an element is added to the stack, it is added to the top, and elements can be deleted only from the top. In other words, a stack is a container where insertion and deletion can occur from only one end, the top of the stack.

Key Operations in a Stack:

1. Push Operation

The push operation adds an element to the top of the stack.

Algorithm for Push Operation:

Step 1: Check if the stack is full (overflow condition).
Step 2: If the stack is not full, increment the top pointer to point to the next empty space.
Step 3: Add the new element at the position pointed to by the top pointer.
Step 4: End.

2. Pop Operation

The pop operation removes the top element from the stack.

Algorithm for Pop Operation:

Step 1: Check if the stack is empty (underflow condition).
Step 2: If the stack is not empty, access the element pointed to by the top pointer.
Step 3: Decrement the top pointer to remove the top element.
Step 4: Return the removed element.
Step 5: End.
C Python Java 
#include<stdio.h>
int stack[4],max=4,top=-1;
int push(){
    int val;
    if (top!=max-1){
        printf("\nEnter an element: ");
        scanf("%d",&val);
        stack[++top]=val;
    } 
    else
        printf("\nOverflow");
    printf("\n");
}  
int pop(){
    if (top!=-1)
        top--;
    else
        printf("\nUnderflow");
    printf("\n");
}
int display(){
    if (top != -1){
        printf("\nStack is:\n");
        for (int i = top; i >= 0; i--)
            printf("%d\n", stack[i]);
    } 
    else
        printf("\nStack is empty");
    printf("\n");
}  
int main(){
    int c;
    while(c!=4){
        printf("\nMenu driven:\n");
        printf("1. Push\n");
        printf("2. Pop\n");
        printf("3. Display\n");
        printf("4. Exit\n");
        printf("\nEnter your choice: ");
        scanf("%d",&c);
        switch(c){
            case 1:
            push();
            break;
            case 2:
            pop();
            break;
            case 3:
            display();
            break;
            case 4:
            printf("\nExit\n");
            break;
            default:
            printf("\nInvalid");
            break;
        }
    }
}
